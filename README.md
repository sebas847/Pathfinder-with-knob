## Specs
Brain: Seeed Studio XIAO RP2040.

PCB Dimentions: 62.53 mm * 77.96 mm

Inputs: 3 Mechanics switches + 1 Encoder EC11 with bttn.

Outpots: 3 LEDs 

Voltage: 3.3V.
